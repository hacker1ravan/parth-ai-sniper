# Parth AI Sniper Pro

AI-powered NIFTY Options Trading Assistant based on Smart Money Concepts (SMC).

## Features

- Market Structure (HH, HL, LH, LL)
- BOS & CHOCH Detection
- Fair Value Gap (FVG)
- Order Block Detection
- Liquidity Sweep Detection
- EMA 20 / EMA 50 Filter
- NSE Option Chain Analysis
- AI Confidence Score
- Entry, Stop Loss & Targets
- Mobile Dashboard
- Telegram Alerts
